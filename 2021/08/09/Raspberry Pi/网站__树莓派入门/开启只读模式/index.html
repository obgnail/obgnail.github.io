开启只读模式

用官方Raspberry Pi OS是最方便开启只读模式的，

在“偏好设定”里找“Raspberry Pi 设定”，在第三个Tab的Overlay File System右侧点 Configure，然后启用 Overlay，Boot Partition 也设为 Read-only ，完工。

命令行也可以配置只读模式，运行sudo raspi-config，选第四项，在里面再选Overlay File System，接下来会出现对话框，选择开启Overlay和Boot Partition只读，接下来就会更新、重启、生效

网上看到的文章提到，只读模式确实能延长存储卡寿命，但限制确实不小，尤其是“所有的改动会在系统重启后消失，因为它们没有写入SD卡里”，如果这样的话，好像“只读”模式确实有点“极端”了。
