最后，扩展说明：

有很多同学会发现，安装完成之后，用node –v查看，还是老版本，安装未生效。

原因：
n 切换之后的 node 默认装在 /usr/local/bin/node，先用 which node 检查一下当前使用的 node 是否是这个路径下的。如上缘由，一般都是因为当前版本指定到了其他路径，更新下/etc/profile文件指定即可。轻松解决。

删除node_modules的缓存，然后重新下载

使用theme

示例

  1. 安装解析器npm install hexo-renderer-less --save;pm install hexo-renderer-pug --save
  2. 在theme目录下执行git clone git@github.com:lotabout/hexo-theme-noise.git
  3. mv hexo-theme-noise noise
  4. Edit _config.yml in hexo root, change theme to noise.

其他的hexo-config和theme-config直接看官方文档即可.

注意

因为hexo有自己的类似于{{}},{%%}等特殊字符.因此,在md文件中,如果要使用到这些字符,必须放在反引号,代码块中.否则使用hexo s和hexo g都编译不了.
