最后，扩展说明：

有很多同学会发现，安装完成之后，用node –v查看，还是老版本，安装未生效。

原因：
n 切换之后的 node 默认装在 /usr/local/bin/node，先用 which node 检查一下当前使用的 node 是否是这个路径下的。如上缘由，一般都是因为当前版本指定到了其他路径，更新下/etc/profile文件指定即可。轻松解决。
