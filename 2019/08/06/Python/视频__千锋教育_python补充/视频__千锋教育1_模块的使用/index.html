OS模块

  2. os.environ:
    操作系统的环境变量

  4. os.curdir:
    获取当前目录

  6. os.listdir(r’D:\trainingfile’):
    以列表形式返回指定目录下的所有文件

  8. os.rmdir(‘hyl’):
    删除目录

  11. os.remove(‘hyl.txt’):
    删除文件

  12. os.system():
    运行命令

  13. os.path.abspath(‘.liangbo’):
    获取当前的绝对路径

    拼接路径

  15. os.path.split(p1):

  17. os.path.isdir:
    是否为目录

  18. os.path.isfile(p1)
    判断文件是否存在

  19. os.path.exists()
    判断文件是否存在

  20. os.path.getsize(path3):

  21. 获取文件名

递归

方式:

  1. 写出基线条件
  2. 找这一次和上一次的关系
  3. 假设当前函数已经能用,==调用自身计算上一次的结果==,再求出本次的结果

如果要用深度变量,一般都是使用==栈==,

思路:每层压入,然后弹出

  1. 压入A,弹出A(弹出的同时获取A下面的文件夹B和C)
  2. 压入BC
  3. 弹出B(弹出的同时获取B下面的文件夹D和E)
  4. 压入DE
  5. 弹出D(弹出的同时获取D下面的文件夹H和I)
  6. 弹出H
    所以现在的栈就是[c,e,i](i为栈顶)
  7. 弹出I
  8. 弹出E(弹出的同时获取E下面的文件夹J和K)
  9. 弹出J
  10. 弹出K
  11. 弹出C(弹出的同时获取C下面的文件夹F和G)
  12. 弹出F
  13. 弹出G

时间模块:

  1. 时间戳:
    以整形或浮点型表示时间的一个以秒为单位的时间间隔
  2. 元组:
    这个元组有9个整形内容:
    (year,month,day,hours,minutes,seconds,weekdey,julia day,DST)
  3. 格式化字符串

  1. time.time():
    返回当前时间的时间戳(浮点数形式)

  2. time.gmtime:
    将时间戳转为UTC时间元组:

  3. time.localtime():
    转为当地时间的元组:

  4. time.mktime:
    将本地时间转为时间元组

  5. time.asctime():
    将时间元组转成字符串

  6. time.ctime():
    将时间戳转为字符串

  7. time.strftime():
    将时间转成特定格式的字符串:

  8. time.strptime()
    将时间字符串转为时间元组

  1. time.sleep():
    延迟时间
  2. time.clock:
    返回当前程序的cup执行时间

datetime是基于time进行了封装,提供了更为使用的函数.接口更直观,更容易调用.

datetime的五个类:

  1. datetime:
    同时有日期和时间
  2. timedelta
    主要用于计算时间的跨度
  3. tzinfo
    时区相关
  4. time
    只关注时间
  5. date
    只关注日期
  1. datetime.datetime.now():
    获取当前时间
  2. datetime.datetime(1999,9,9,10,28,25,123456):
    获取指定时间
