绪论

  • 平台性是操作系统的重要特点。
  • 操作系统在计算机中的主要特点是管理硬件及资源。

操作系统的要求 :

  • 公平性对于操作系统很重要，是操作系统较核心的理念。
  • 操作系统的开发，效率仍然是其追求的目标。
  • 操作系统本身要保持稳定的特性。
  • 安全

操作系统的目标 :

  • 方便
  • 硬件的高效利用
  • 整体可靠、稳定、安全
