  • 查看命令在哪个位置，我们需要使用which；
  • 而对于命令的具体参数及使用方法，我们需要用到强大的man；

2. 文件及目录管理

文件管理不外乎文件或目录的创建、删除、查询、移动
文件查询是重点，用find来进行查询；find的参数丰富，也非常强大；

注意win的路径使用反斜杠,Linux使用斜杠.

2.1. 创建和删除

  • 创建目录：mkdir
    创建多级目录需要使用-p选项(例如:mkdir liangbo/dir1/dir2)
  • 创建文件:touch
  • 删除目录或文件：rm
    (删除文件：rm 1.txt)
    (删除目录：rm -rf dir)
    (删除日志 rm log (等价: $find ./ -name “log” -exec rm {} ;))
  • 移动：mv
  • 复制：cp (复制目录：cp -r )

有时候，需要给文件创建一个别名，我们需要用到ln，使用这个别名和使用原文件是相同的效果；

2.2. 目录切换

  • 找到文件/目录位置：cd
  • 切换到上一个工作目录： cd -
  • 切换到home目录： cd or cd ~
  • 显示当前路径: pwd
  • 更改当前工作路径为path: $cd path

2.3. 列出目录项

  • 显示当前目录下的文件 ls
  • 按时间排序，以列表的方式显示目录项 ls -lrt
