使用帧动画:

动画:

  • transform:
    CSS3中的形变处理,实现文字和图像的旋转,缩放,倾斜和移动等.
  • transitions:
    CSS3中动画功能,通过一个属性值平滑够毒到另一个属性值来实现
  • animation:
    CSS3中的动画功能,通过在样式中创建多个关键帧,在这些关键帧中编写样式,并且能够在页面上综合运行这些关键帧来实现较为复杂的动画.
