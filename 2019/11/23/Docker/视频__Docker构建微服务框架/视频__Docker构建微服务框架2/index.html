Swarm 的架构

swarm 的节点有两种角色

  • manager : 集群的大脑(为了避免单点故障,大脑有多个)
  • worker : worker部署容器可能需要数据同步
  • 既然大脑有多个 , 那么就涉及到大脑的同步 . 这里的同步就使用了Raft协议
  • worker使用Gossip网络进行信息同步
