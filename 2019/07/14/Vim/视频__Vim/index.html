Vim

  1. 不保存退出: :q!

  2. hjkl分别代表左下上右

  3. w(word):下一个单词

  4. b(back):上一个单词

  5. Ctrl+f(forward):下一页

  6. ctrl+d:上一页

  7. 88gg:跳转到第88行

  8. 77k:往上跳77行

  9. 30j:往下跳30行

  10. s/node:查找node

    n:查找后挨个寻找下一个node

    N:查找后挨个寻找上一恶搞node

  11. dd:删除当前行

  12. d3d:删除当前行开始的三行(或者3dd)

  13. cc:剪切当前行(自动跳转为编辑模式)

  14. c3c剪切当前行开始的3行

  15. yy:复制

  16. p:粘贴

  17. u(undo):撤销

  18. G : 到文件尾

  19. gg : 到文首

  20. ma :在光标处标记一个书签a（必须小写）

  21. `a 到书签a处（注意，这是1左边那个键）

  22. * 当光标停留在一个单词上，* 键会在文件内搜索该单词，并跳转到下一处；

  23. # 当光标停留在一个单词上，# 在文件内搜索该单词，并跳转到上一处；

  24. o 在光标所在行的下一行增添新行

  25. O 在光标所在行的上一行增添新行
