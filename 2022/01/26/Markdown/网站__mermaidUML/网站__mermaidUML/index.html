流程图

方位

  • TB - top bottom
  • BT - bottom top
  • RL - right left
  • LR - left right
  • TD - same as TB

连接

命令 形状 添加文本
A1 –> B1 直线箭头 –text–>
A2 — B2 直线 –text—
A3 -.-> B3 虚线箭头 -.text.->
A4 -.- B4 虚线 -.text.–
A5 === B5 加粗直线 ==text===
A6 ==> B6 加粗直线箭头 ==text==>

使用 & 符号 连接，来减少重复项

转义的字符可以使用 "..." 双引号来处理

注释 %% 开始忽略任何语法

样式

  • style node style 节点样式
  • classDef className style声明类
    • class node1,node2 className; 样式附加方式 1
    • node:::className 直接在流程中附加样式
  • style 描述 svg 样式
  • , 来分割
  • ；fill 填充色，
  • stroke 线条
  • stroke-width 线条宽度
  • color 前置色。

参与者

消息

  • ->> 显示为实线箭头（主动发出消息）
  • -->> 显示为虚线箭头（响应）
  • -x 显示为末尾带「X」的实线箭头（异步消息）

激活框

从消息接收方的时间线上标记一小段时间，表示对消息进行处理的时间间隔。

箭头符号后的 + 和 - 的使用方法和效果，它们相当于激活框的开关

类

  • class Name{} ：命名类
  • +Attr： 属性（在括号内）
  • +Fun()： 函数（在括号内）
  • class1 <|-- class2 关系隶属

属性方法的权限

  • + Public
  • - Private
  • # Protected
  • ~ Package/Internal

类型

  • <<Interface>> To represent an Interface class
  • <<abstract>> To represent an abstract class
  • <<Service>> To represent a service class
  • <<enumeration>> To represent an enum
